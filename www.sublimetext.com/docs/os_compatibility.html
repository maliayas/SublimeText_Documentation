Operating System Compatibility

Version:
Dev 3.2 3.1 3.0

Sublime Text runs on the Windows, Mac and Linux operating systems. It supports x86 (32bit) and x86_64 (64bit) processors. ARM processors are not supported.

32bit builds of Sublime Text do not include Git Integration.

Windows

  • Windows 10
  • Windows 8/8.1
  • Windows 7
  • Windows Vista
  • Windows XP

Mac

Only 64bit builds are available on Macs.

  • macOS 10.12+
  • OS X 10.9+
  • OS X 10.7+

Linux

Only 64bit builds are available on Linux.

Linux installs require the following libraries:

  • GLIBC 2.15 (libc.6.so)
  • GLIBC 2.12 (libc.6.so)
  • GTK 3.10+ (libgtk-3.so, libgdk-3.so)
  • GTK 2 (libgtk-x11-2.0.so, libgdk-x11-2.0.so)
  • GLib 2 (libgio-2.0.so)
  • FreeType 2 (libfreetype.so)
  • Cairo (libcairo.so)
  • Pango (libpango-1.0.so)
  • Pangocairo (libpangocairo-1.0.so)
  • OpenGL (libGL.so.1)
